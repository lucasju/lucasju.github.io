Installation

  • Première étape

    Menu extexion

    Rendez vous sur l’administration de votre site WordPress. Allez sur le menu Extension, et cliquez sur Ajoutez.

  • Deuxième étape

    Recherche du plugin

    Dans le champs (à droite de l’écran) « Chercher parmi les extensions » entrez « Allthecontent » et valider. Repérez le plugin « Allthecontent » et cliquez sur le bouton « Installez maintenant ».

  • Troisième étape

    Installation du plugin

    Si demandé, completez les informations de connexion de votre FTP, puis continuez. Patientez pendant l’installation du plugin, puis cliquez sur « activez l’extension ».

  • Quatrième étape

    Réglage du plugin

    Sur le panneau latéral gauche ouvrez le menu déroulant au niveau « Allthecontent » puis cliquez sur « Settings ». Renseignez le chemin FTP vers le répertoire accessible de votre WordPress où le flux vous sera livré.
